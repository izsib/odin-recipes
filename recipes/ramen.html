Ramen :)


Description

yum, Ramen! Everyone loves noodles, right? It's a great way to feed a yourself and a perfect dish to eat on the couch."

Ingredients

  • ramen noodle packet
  • water
  • an egg if you're feeling extravagant

Steps

  1. boil water in a pan
  2. add noodles
  3. wait 3 min
  4. add egg if using
  5. add seasoning packet
  6. stir
  7. add to a bowl and enjoy
